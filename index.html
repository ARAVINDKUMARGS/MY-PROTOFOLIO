03 / Projects

Things I've built

L0 · Hardware

STM32 Peripheral Driver Suite

Problem: most embedded intros stop at HAL calls. I wanted to know what a HAL function actually does underneath.

Built four peripheral drivers — GPIO, SPI, I2C, USART — entirely through direct register programming on an STM32F407G-DISC1 (ARM Cortex-M4), with zero HAL. Implemented both polling and interrupt-driven modes, hand-configuring NVIC, EXTI, and RCC registers.

STM32F407 Embedded C ARM Cortex-M4 NVIC / EXTI / RCC
L1 · Systems

Nand2Tetris — A Computer from First Principles

Problem: understand what actually happens between a NAND gate and a running program, with nothing taken on faith.

Built a complete computer system bottom-up — ALU, CPU, memory hierarchy, assembler, VM translator, compiler, and a simple OS — starting from a single NAND gate.
